# Demo Guide

## Judge-Friendly Demo Flow

1. Run `make demo-local`
2. Observe deployment summary and tx hashes
3. Observe normal regime swap output
4. Observe forced hard-depeg policy and cooldown block
5. Observe incentives claim output

## Target Commands

- `make demo-normal`
- `make demo-depeg`
- `make demo-incentives`
- `make demo-all`

Each demo mode runs deterministic, on-chain logic only.
